## Microsoft official practice assessment

[Open the official AI-900 practice assessment ->](https://learn.microsoft.com/credentials/certifications/azure-ai-fundamentals/)

- 50 questions, drawn from a larger pool - repeat as needed.

## How to use it

1. **First attempt** - closed book, timed. Treat it like the real test.
2. Note every wrong answer. For each one, identify which **page in this guide** covers the topic and re-read it.
3. **Second attempt** - different question pool. Aim for 90%+.
4. Move on to the **AI Copilot Quiz** page for unlimited practice across multiple AI assistants.
